VK — Полное исследование

Бесплатные инструменты

  • ✅ VK API v5 (стандарт) — посты, комментарии, подписчики
  • ✅ VK Stats (встроенный аналитик) — охват, ER, демография аудитории (только для владельцев сообществ)
  • ✅ VKSTAT.ru (парсинг + ML) — рейтинг сообщества, сравнение с конкурентами, динамика подписчиков, прогноз роста

Платные инструменты (+3-5x больше информации)

Инструмент Стоимость Возможности
HypeAuditor / VKSPY $49-150/мес Детальная демография аудитории, аудит аккаунта, поиск целевой аудитории, сравнение с конкурентами, контент-план
Popsters $99/мес Полная аналитика VK, Telegram, Instagram одновременно, интеграция с CRM, предсказание виральности
LiveDune $49/мес Управление контентом, аналитика всех каналов, примеры лучшего контента конкурентов
Brand Analytics от 30 000 ₽/мес Глубокий анализ конкурентной среды в VK, мнения аудитории, упоминания брендов

Максимум информации по VK:

  • Профиль подписчиков (возраст, пол, регион, интересы, доход)
  • Вовлечённость по типу контента
  • Сезонность, часы пиковой активности
  • Воронка конверсии (с пиксельной отслеживанием)
  • Сравнение с конкурентами в реальном времени

Telegram — Исследование

Бесплатные инструменты

  • ✅ TGStat.com API (бесплатный тир, ограничено) — статистика каналов, подписчики, ER
  • ✅ Telethon (Python lib) — контент, комментарии, юзеры в чатах
  • ✅ Statistic (встроенный для каналов) — стандартная аналитика (только свой канал)

Платные инструменты (+300% информации!)

Инструмент Стоимость Возможности
TGStat Pro $99/мес История всех постов, прогнозы виральности, рубрики контента, целевая аудитория, сравнение конкурентов, граф аудитории
Telemetr Pro $99/мес Глубокая аналитика, предложения по контенту, прогноз ER
Radarbot $199/мес AI-анализ аудитории, контент-рекомендации, инструменты для рекламы

Максимум информации по Telegram:

  • История всех напостов за 1+ год (текст, видео, медиа)
  • Карта аудитории (какие ещё каналы она читает)
  • Демография (примерно: возраст, геолокация, интересы)
  • Тренды внутри аудитории (что обсуждают)
  • Лучшее время для постинга, типы контента с максимальным ER

Instagram — Исследование

Бесплатные инструменты

  • ✅ Instaloader — посты, сторис, видео (при авторизации)
  • ✅ Instagram Insights (встроенный) — только если владелец бизнес-аккаунта
  • ⚠️ Instagram Graph API (теоретически) — практически недоступна для конкурентов

Платные инструменты (+500% информации!)

Инструмент Стоимость Возможности
HypeAuditor $49/мес Аудит аккаунта, поддельные подписчики, демография, контент-анализ, бенчмарки
Sprout Social $249/мес Управление, аналитика, AI-рекомендации, коллаборации
Buffer / Later $99/мес Планирование, аналитика, сравнение каналов
Iconosquare $99/мес Детальная аналитика, демография, контент-рекомендации
Phlanx €99/мес Анализ конкурентов, поиск целевой аудитории, фейк-детектор
Influee / Instazood $20-100/мес Демография, ER по типам контента

Максимум информации по Instagram:

  • Демография аудитории (возраст, страна, пол, интересы)
  • % подписчиков, которые потенциально фейк
  • Охват + ER по каждому посту
  • Лучшее время постинга
  • Типы контента с максимальной конверсией
  • Коллаборации (с кем сотрудничают конкуренты)

Платные инструменты (+1000% информации о рекламе конкурентов!)

Инструмент Стоимость Возможности
